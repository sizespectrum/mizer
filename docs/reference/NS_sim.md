# Example MizerSim object for the North Sea example

A MizerSim object containing a simulation with historical fishing
mortalities from the North Sea, as created in the tutorial "A
Multi-Species Model of the North Sea".

## Usage

``` r
NS_sim
```

## Format

A MizerSim object

## Source

<https://sizespectrum.org/mizer/articles/a_multispecies_model_of_the_north_sea.html>

## See also

Other example parameter objects:
[`NS_params`](https://sizespectrum.org/mizer/reference/NS_params.md)

## Examples

``` r
plotBiomass(NS_sim)
```
