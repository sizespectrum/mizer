# Units string for the flux returned by [`getFlux()`](https://sizespectrum.org/mizer/reference/getFlux.md)

Units string for the flux returned by
[`getFlux()`](https://sizespectrum.org/mizer/reference/getFlux.md)

## Usage

``` r
flux_units(power)
```

## Arguments

- power:

  The power of weight the flux was multiplied by.

## Value

A character string with the units of the flux.
