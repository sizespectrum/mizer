# Summary plot for `MizerSim` objects

After running a projection, produces 5 plots in the same window: feeding
level, abundance spectra, predation mortality and fishing mortality of
each species by size; and biomass of each species through time. This
method just puts the plots generated by
[`plotBiomass()`](https://sizespectrum.org/mizer/reference/plotBiomass.md),
[`plotFeedingLevel()`](https://sizespectrum.org/mizer/reference/plotFeedingLevel.md),
[`plotSpectra()`](https://sizespectrum.org/mizer/reference/plotSpectra.md),
[`plotPredMort()`](https://sizespectrum.org/mizer/reference/plotPredMort.md)
and
[`plotFMort()`](https://sizespectrum.org/mizer/reference/plotFMort.md)
all in one window.

## Usage

``` r
# S3 method for class 'MizerSim'
plot(x, ...)
```

## Arguments

- x:

  An object of class
  [MizerSim](https://sizespectrum.org/mizer/reference/MizerSim-class.md)

- ...:

  Arguments passed on to the individual plotting functions
  [`plotBiomass()`](https://sizespectrum.org/mizer/reference/plotBiomass.md),
  [`plotFeedingLevel()`](https://sizespectrum.org/mizer/reference/plotFeedingLevel.md),
  [`plotSpectra()`](https://sizespectrum.org/mizer/reference/plotSpectra.md),
  [`plotPredMort()`](https://sizespectrum.org/mizer/reference/plotPredMort.md)
  and
  [`plotFMort()`](https://sizespectrum.org/mizer/reference/plotFMort.md).

## Value

A viewport object
