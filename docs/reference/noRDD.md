# Give density-independent reproduction rate

Simply returns its `rdi` argument.

## Usage

``` r
noRDD(rdi, ...)
```

## Arguments

- rdi:

  Vector of density-independent reproduction rates \\R\_{di}\\ for all
  species.

- ...:

  Not used.

## Value

Vector of density-dependent reproduction rates.
